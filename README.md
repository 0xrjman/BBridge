
# Platdot

## Installation

### Prerequisites

- Make sure the `Go` environment is installed

### Building

`git clone https://github.com/RJman-self/Platdot.git`

`make build`: Builds `platdot` in `./build`.

**or**

`make install`: Uses `go install` to add `platdot` to your `GOBIN`.
